# Stock-Market-
This project demonstrates an end-to-end stock market data analytics workflow using Python, SQL, and Power BI to support data-driven financial insights.
Project Overview

This internship project focuses on analyzing stock market price data to understand price movements, trading volume, and overall stock performance using Python, SQL, and Power BI.

Objectives

Clean and preprocess raw stock market data

Analyze price trends using open, high, low, and close values

Study trading volume patterns

Visualize insights through interactive dashboards

Tools & Technologies

Python – data cleaning, preprocessing, and validation

SQL – querying and aggregating stock data

Power BI – interactive dashboards and visual reporting

Methodology

Cleaned and standardized raw data using Python

Converted date and numeric fields and removed invalid records

Performed analysis using SQL queries

Built Power BI dashboards to visualize price and volume trends

Key Insights

Cleaned data improves accuracy of financial analysis

OHLC values help identify stock price trends

Trading volume provides insight into market activity
